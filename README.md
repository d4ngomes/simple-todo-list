## Como executar

```bash
# clonar repositório
git clone https://github.com/d4ngomes/simple-todo-list

# entrar na pasta do projeto
cd simple-todo-list

# instalar dependências
npm i

# executar json-server
json-server --watch todo.db.json

# executar o projeto
ng serve --open
```
